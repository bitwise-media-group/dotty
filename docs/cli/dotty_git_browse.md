## dotty git browse

Open the upstream (else origin) repository page in a browser.

### Synopsis

Opens the forge homepage for the upstream remote when present, otherwise
origin.

```
dotty git browse [flags]
```

### Examples

```
  dotty git browse
```

### Options

```
  -h, --help   help for browse
```
