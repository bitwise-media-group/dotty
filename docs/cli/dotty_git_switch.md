## dotty git switch

Pick a stack layer and check it out.

### Synopsis

Presents the current stack's layers in a fuzzy picklist and checks
out the chosen one.

```
dotty git switch [flags]
```

### Examples

```
  dotty git switch
  dotty git stack     # alias
```
