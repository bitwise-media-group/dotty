## dotty git down

Move toward the trunk of the current stack.

### Synopsis

Checks out the layer [num] hops toward the trunk (default 1) and
prints the stack.

```
dotty git down [num] [flags]
```

### Examples

```
  dotty git down
  dotty git down 2
```
