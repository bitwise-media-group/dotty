## dotty git up

Move toward the tip of the current stack.

### Synopsis

Checks out the layer [num] hops toward the stack tip (default 1) and
prints the stack.

```
dotty git up [num] [flags]
```

### Examples

```
  dotty git up
  dotty git up 2
```
